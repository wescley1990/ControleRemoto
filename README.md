# CONTROLE REMOTO
Aprensentação de um Controle Remoto na linguagem JAVA - Poo

## Funcionalidades

Oferecendo as seguintes funcionalidades:

- Liga, Desliga, Menu, Volume, Mudo, Play e Pause;
- interface Controladora 
- getters/setter
- metodos abstratos
- metodos especiais
- implementação de interface
- construtores
